Actions

Quick Actions:

Full Connection: Establishes complete connection sequence automatically

Disconnect All: Closes all active connections

Step-by-Step Actions

Control: Establish control WebSocket connection

Tunnel: Send tunnel message (requires Control connection)

Relay: Connect to MeshRelay (requires Control and Tunnel)

Instructions

Setup:

  1. Enter your OpenFrame API URL (e.g., https://openframe.example.com/tools/meshcentral)
  2. Provide your OpenFrame authentication token
  3. Provide your MeshCentral authentication token
  4. Provide your MeshCentral relay authentication token
  5. Replace 'your-node-id-here' in the buttons with actual node IDs

Usage:

  • Full Connection: Use this for automatic setup of all connections
  • Step-by-Step: Use individual buttons for debugging or manual control
  • Monitor the connection status panel for real-time updates
  • Check browser console for detailed logging
